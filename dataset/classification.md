# Auto classifications

## wordPatternFrequencyTable

Contains different patterns and the number of occurences,  e.g.

```json
[
    {
        "pattern": "[WORD][number]", 
        "occurrences": 46
    }
]
```

## patternFrequencyTable

```json
[
    {
        "pattern": "AA99", 
        "occurrences": 46
    }
]
```
